# How to Deploy
1. Create Instance in GCP

| Requirements  | Spec            | 
| ------------- |:---------------:| 
| CPU           | 2 vCPUs         |
| RAM           | 4 GBs           | 
| Storage       | 10 GBs          |
| OS            | Ubuntu 20.04 LTS|  

2. Install Docker

```bash
apt update
apt install -y docker.io && docker-compose
```

3. Git Clone

```bash
git clone https://github.com/poonawitch/Prayuth.git
```

4. Go to Prayuth folder

```bash
cd prayuth
```

5. Use Docker command

```bash
docker-compose up -d
```

6.Open your browser

http://localhost:8888
